About Game

  • At a time only one player is allowed to make a move, player 1 will start the game.
  • Once a player has initiated his/her move, next player will given a turn to make a move
  • Every a time button is clicked score will be incremented by some random integer value within a range of 1 to 10
  • Player whose score crosses the target value will be declared as the winner.
  • Target value is being set to 100.
  • If you want to play the game again then simply press reset button and you can start all over again.

Have Fun!!(Although it's a boring game but still)
